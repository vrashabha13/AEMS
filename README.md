# Advanced Examination Management System (AEMS)

AEMS is a comprehensive web-based platform designed to facilitate secure and efficient online examinations. By integrating advanced technologies, it offers a seamless experience for educators and students alike.

## Key Features

- **Automated Grading:** Utilizes AI algorithms to evaluate answers, providing immediate and accurate results.
- **Plagiarism Detection:** Ensures academic integrity by identifying potential instances of content duplication.
- **Classroom Management:** Organizes students into virtual classrooms, simplifying exam administration and monitoring.
- **Randomized Quizzes:** Enhances assessment security by generating unique question sets for each examinee.
- **PDF Support:** Allows for the import and export of exam materials and results in PDF format, facilitating easy distribution and record-keeping.

## Technology Stack

- **Frontend:** Developed using React and TypeScript to create a dynamic and type-safe user interface.
- **Backend:** Built with Node.js, ensuring scalable and efficient server-side operations.
- **AI Integration:** Incorporates Python-based services leveraging the Google Gemini API for enhanced functionalities such as automated grading and plagiarism detection.

## Getting Started

To set up and run AEMS locally:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/vrashabha13/AEMS.git
   ```

2. **Navigate to the Project Directory:**
   ```bash
   cd AEMS
   ```

3. **Install Dependencies:**
   ```bash
   npm install
   ```

4. **Set Up Environment Variables:**
   Create a `.env` file in the root directory and configure the necessary environment variables as specified in `.env.example`.

5. **Start the Development Server:**
   ```bash
   npm run dev
   ```

   The application will be running at `http://localhost:5173`.
